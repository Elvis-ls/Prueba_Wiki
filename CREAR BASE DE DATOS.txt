instalar dependencias necesarias: npm install express --save
dependencias de desarrollo: npm install -D typescript ts-node @types/node @types/express prisma

npm install prisma --save-dev : para instalar prisma.
npm install @prisma/client : instalar el cliente de prisma.
npx prisma generate : Para generar el cliente.
npx prisma init : crear la carpeta prisma y schema.prisma
npx prisma migrate dev --name "Tunombre" : Para empezar la migración y creación de la tabla.
npx prisma studio : Para ver si esta trabajando con mi base de datos y ver los datos.
npx prisma migrate reset : Para eliminar la migración.

http://localhost:3001/api/validate/validateUser :Ruta para verificar los 4 campos de seguridad.
http://localhost:3001/api/auth/login :Ruta de login user
http://localhost:3001/api/register/:cedula/complete :Ruta para completar la información de registro.
http://localhost:3001/api/user/information :Ruta para actualizar la información en el inicio de la pagina.
http://localhost:3001/api/user/recovery-password :Ruta para recuperar contraseña user.
http://localhost:3001/api/user/monto-contribution :Ruta para que el user agregue un monto.
http://localhost:3001/api/user/payment :Ruta para que se actualice el después de realizar su pago.
http://localhost:3001/api/user/mark-as-read/:idnotification :Ruta para marcar como leida una noticia.
http://localhost:3001/api/user/create/credits :Ruta para crear un credito.

Rutas Admin
http://localhost:3001/api/admin/create-user :Ruta para que el administrador registre un user como accionista.
http://localhost:3001/api/admin/create :Ruta para crear un administrador.
http://localhost:3001/api/admin/login : Ruta login del administrador.
http://localhost:3001/api/admin/create-news :Ruta para agregar una noticas solo admin.
http://localhost:3001/api/admin/shareholder-form/:id :Ruta para aceptar o rechazar el formulario de shareholder.
http://localhost:3001/api/admin/recovery-password :Ruta para recuperar contraseña Admin.

Ruta shareHolder formulario
http://localhost:3001/api/formulario :Ruta para crear el formulario.
